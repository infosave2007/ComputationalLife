**Одна идея, доведённая до строгости:** саморепродукция, самонаблюдение и
самомодификация — это не метафоры, а классические теоремы. Вместе с универсальным
вычислением и универсальным предсказанием они образуют единую **вычислительную теорию
жизни и разума**. Каждое заголовочное утверждение здесь *реализовано, запущено,
самопроверено ассертами и покрыто независимым тестом*. Физика не используется и не нужна;
единственная честная точка контакта с NVG-физикой — это *граница* (потолки ресурсов),
описанная в [`NVG_INTERFACE.md`](NVG_INTERFACE.md). Каждое утверждение сопоставлено с
конкретной функцией и тестом в [`CLAIMS.md`](CLAIMS.md).

## Объединяющая картина

| Слой | Кто | Что | Модуль |
|---|---|---|---|
| саморепродукция | **фон Нейман** (1948) | конструктор + двойного назначения лента-описание | [`replication`](complife/replication.py) |
| универсальное вычисление | **Тьюринг** (1936) | одна машина вычисляет всё вычислимое | (субстрат) |
| самореференция | теорема рекурсии **Клини** | программа может читать *и переписывать* свой код | [`self_reference`](complife/self_reference.py) |
| самонаблюдение | **Конант–Эшби** (1970) | хороший регулятор обязан *моделировать* систему | [`self_model`](complife/self_model.py) |
| предсказание = сжатие | **Соломонов / Хаттер** | лучшая модель — это кратчайшая программа | [`self_model`](complife/self_model.py), [`physical_limits`](complife/physical_limits.py) |

Повторяющаяся архитектура — **«дешёвая программа + мощный общий декодер»**: сложность
живёт в декодере и во входных данных, а не в программе. Именно это и есть компрессор
(модель + кодер), геном (рецепт + физика/химия) и самомодель агента (сжатый предиктор
самого себя).

## Четыре модуля

### `self_reference` — самореференция реальна, *обща* и дёшева
- Проверенный Python-**квайн** (stdout == исходник, байт в байт): **75 байт**; плюс
  проверенные квайны в игрушечных языках (**Underload `(:aSS):aSS`**, 10 символов;
  минимальный λ-квайн на Lisp, 99 байт).
- **Вторая теорема рекурсии, конструктивно**: программы, чей вывод — *нетривиальная
  функция собственного исходника* (`sha256`, `длина`, `реверс`, `upper`), каждая
  проверена в подпроцессе. Плюс **самомодифицирующаяся** программа, печатающая свою копию
  с инкрементированным счётчиком при неизменных остальных байтах. Это и есть «читать *и
  переписывать* свой код» — теперь показано, а не заявлено.
- **Аддитивная константа накладных расходов, двумя честными способами:** у repr-конструкции
  накладные расходы измеренно равны **159 + (число экранированных символов)** — то есть
  `O(#escapes)`, константа только для payload без спецсимволов; у **content-agnostic**
  конструкции (payload + фиксированный самочитающий хвост) накладные расходы **ровно
  постоянны (76 байт)** для *любых* байтов, включая кавычки/бэкслэши/переводы строк.

### `self_model` — самомодель *обязана* быть сжатием с потерями
- Точное самовключающее моделирование **невозможно** (`2^{b_M} < |T|`, если цель содержит
  модель плюс что-то ещё) — проверено на 5 режимах.
- Пол самонепредсказуемости **`H(T|M) ≥ H(T) − b_M`** показан для **произвольной
  детерминированной модели `g`** (случайное хеширование в `2^{b_M}` корзин), а не только для
  лучшего случая (префиксной модели) — граница сводится к `I(T;M) = H(M) ≤ b_M`, что мы
  меряем напрямую, с **бутстрап-доверительными интервалами** и на источниках
  **равномерном, Zipf и Марковском**. Расточительная `g` сидит строго выше пола (настоящая
  нижняя граница).
- Теорема **Конанта–Эшби продемонстрирована**, а не процитирована: перебор всех
  детерминированных политик показывает, что оптимальный регулятор системы `Z = (D+R) mod n`
  *вынужден* быть моделью `ρ(D) = (c − D) mod n` (единственной с точностью до цели), а у
  регулятора на `b` бит остаточная вариативность **`H(Z) = H(D) − b`** — точный аналог пола
  самомодели.

### `replication` — маленькие саморазвивающиеся копии, с пределом
- **Лента двойного назначения:** организм = машинерия + лента, используемая двояко —
  *интерпретируется* для сборки машинерии и *копируется дословно* потомку. `child == parent`
  (байт в байт), `grandchild == child` (потомок самозагружается из собственных байтов),
  мутация payload **наследуется дословно**; отображение родитель→потомок — оператор `O(1)`,
  не зависящий от длины payload.
- **Порог ошибок Айгена, тремя способами:** замкнутая форма (2 компартмента); **полное
  квазивидовое распределение** как собственный вектор Перрона–Фробениуса точно свёрнутой по
  классам Хэмминга матрицы отбор–мутация `W = Q·diag(f)` — **сверено с полным перебором `2^L`
  до `~10⁻¹⁶`**; и точный конечно-популяционный прогон Райта–Фишера. Все трое согласны, что
  **`μ_crit·L → ln σ`** (приближается *снизу* с ростом `L`). Честно: с обратными мутациями
  доля мастера *гладкая* и строго положительна при конечном `L` — резкая катастрофа это
  предел `L → ∞`.

### `physical_limits` — единственное место, где входит физика
- **Хранение:** Ландауэр `kB·T·ln2` = **2.87×10⁻²¹ Дж** при 300 K; мозг использует
  **~10⁻²⁷·⁶** своего бекенштейновского потолка — жизнь информационно-*разрежена* (показано
  от E. coli до дата-центра).
- **Скорость (теперь вычислена, а не процитирована):** Марголус–Левитин `4E/h` ≈
  **7.6×10⁵⁰ оп/с** для мозга; Бремерманн `c²/h` ≈ **1.36×10⁵⁰ оп/с/кг**; предельный ноутбук
  Ллойда **5.4×10⁵⁰ оп/с**.
- **Пропускная способность:** при бюджете мозга ~20 Вт / 310 K Ландауэр допускает
  **6.7×10²¹** необратимых стираний/с — мозг работает на ~7 порядков *выше* поминутного
  термодинамического минимума на бит (разрежен по битам, расточителен по энергии на бит).
- **NVG-интерфейс (модельно-зависимый, явно помечен):** при NVG-*входах* `M_Ω = 859 МэВ` и
  «вакуум плавится при `ρ_c = M_Ω⁴/(ℏc)³` = 1.26×10²⁰ кг/м³» экстремальное регулярное ядро
  имеет внутренний масштаб де Ситтера `l = 1.128 км` и — исправленная геометрия —
  экстремальный горизонт при `r_h = √3·l = 1.954 км`, чья энтропия Бекенштейна–Хокинга
  равна **6.6×10⁷⁶ бит** (площадная граница на внутреннем масштабе `S(l) = 2.2×10⁷⁶` — это
  *другая* величина). Физика ограничивает контейнер, но не поставляет вычисление.

## Установка и запуск

```bash
pip install -e .            # или: pip install -r requirements.txt
python -m complife          # запустить все четыре модуля, вывести общую сводку PASS/FAIL
python -m complife 01 03    # запустить только названные модули
```

Четыре нумерованных скрипта по-прежнему работают из корня репозитория и перенаправляют в пакет:

```bash
python3 01_self_reference_quine.py   #  ==  python -m complife.self_reference
```

## Тесты

```bash
make test        # полный набор (квайны в подпроцессах + Монте-Карло), ~8 с
make test-fast   # пропустить медленные Монте-Карло / подпроцессы
make cov         # отчёт покрытия
make lint        # ruff
```

Каждое научное утверждение выше — отдельный `pytest`-тест (см. [`CLAIMS.md`](CLAIMS.md)),
а не только ассерт внутри демо. Заголовочные числа снимаются в [`results/`](results/) на
каждом прогоне.

## Структура проекта

```
complife/            импортируемый, протестированный пакет
  common.py          оценщики энтропии, бутстрап-CI, константы, репортинг
  self_reference.py  квайны, 2-я теорема рекурсии, аддитивная константа
  self_model.py      пол DPI для произвольной g, источники, регулятор Конанта–Эшби
  replication.py     лента двойного назначения, квазивидовой собственный вектор, WF
  physical_limits.py потолки хранения/скорости/пропускной способности, геометрия NVG
  __main__.py        `python -m complife`
tests/               по файлу на модуль + common + smoke + snapshots (51 тест)
01..04_*.py          тонкие запускаемые обёртки (сохраняют исходные точки входа)
CLAIMS.md            каждое утверждение -> функция -> тест -> оговорка
NVG_INTERFACE.md     единственная честная физическая граница (и category errors)
```

Стандартная библиотека + **numpy**. Детерминировано (стохастика на фиксированных seed).

## Честные оговорки

- Это стандартная **алгоритмическая теория информации + кибернетика** (Клини, Конант–Эшби,
  фон Нейман, Айген, Соломонов/Хаттер). Ничто здесь не бьёт Шеннона; ничему не нужна новая
  физика.
- **Сознание здесь *не* выводится.** «Самомодифицирующаяся, самонаблюдающая программа =
  самосознание» — недоказанный мост через трудную проблему (Чалмерс); самореференция
  *почти-необходима* для самомоделирования, но синтаксична — она ни по какому доказательству
  не влечёт феноменального опыта.
- Отношение к NVG-физике — это **граница, а не слияние**, см. [`NVG_INTERFACE.md`](NVG_INTERFACE.md).
  NVG-числа опираются на постулированные модельные входы (масштаб 859 МэВ, механизм плавления)
  и помечены как таковые; потолки Ландауэра/Бекенштейна/голографический/скоростные — из первых
  принципов.
